# base-springboot-starter

#### 介绍
通用的 springboot starter web微服务开发的大部分配置功能都进行了封装,做了统一响应，异常处理，日志记录，链路追踪，限流策略，敏感词过滤策略，缓存策略，线程异常，线程之间传递值的处理，分布式锁注解，支持配置注册MQ生产者已经消费者，声明队列，交换机，对mybatis-plus进行增强，支持记录慢sql日志，支持swaggerv3,支持开放接口的参数校验...

#### 软件架构
软件架构说明：springboot + redis + mysql + mybatis-plus + rabbitmq + nacos+openfeign
